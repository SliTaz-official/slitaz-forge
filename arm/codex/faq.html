Frequently Asked Questions

Sorted questions and short answers.

  • Can I use SliTaz ARM to build a NAS ?
  • How to install new software ?
  • Does SliTaz handle small screens ?
  • Which devices are supported ?

Can I use SliTaz ARM to build a NAS ?

Yes. You need to install the 'nfs-utils' packages for NFS mount, you can also use Rsync or an FTP server.

How to install new software ?

Use the SliTaz packages toolset utilities from the cmdline or the Ncurses dialog interface: 'spk' 'spk-dialog'

Does SliTaz handle small screens ?

Yes, the SliTaz Tiny/Touch Screen desktop provides a nice environment for small displays.

Which devices are supported ?

Actually only the Rasperry Pi is tested by SliTaz developers. But any armel devices should be supported.
