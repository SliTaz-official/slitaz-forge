SliTaz Raspberry Pi

Official SliTaz GNU/Linux Raspberry Pi distribution, a small operating system for a small computer! The goal is to provide a fast, minimal footprint and optimized distro for the Raspberry Pi.

SliTaz Raspberry Pi is a rolling model distro with a selective set of packages.

TazBerry

Sat-rpi builder

The Raspberry Pi has a lot of specifications. SliTaz ARM provides 'sat-rpi' to build custom SliTaz Rpi distro images ready to boot. Quickstart guide is in the SliTaz Raspberry Pi README
